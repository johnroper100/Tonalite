# Keyboard Shortcuts

There are several keyboard shortcuts built into Tonalite to make it easier to do certain actions while programming and/or running shows.

| Key        | Action      |
|:----------:|:-----------:|
| R          | Record Cue  |
| End        | Stop Cue    |
| Page Up    | Next Cue    |
| Page Down  | Last Cue    |
| Home       | First Cue   |
| SHIFT+A    | Add Fixture |
| CTRL+ALT+N | New Show    |
| CTRL+S     | Save Show   |
| CTRL+Z     | Undo        |
| CTRL+Y     | Redo        |

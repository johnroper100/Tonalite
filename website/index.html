ABOUT TONALITE

Tonalite was created to be the first fully network-based control lighting control system for any venue. Tonalite operates over ethernet which allows it to be fully modular. Running from a central server, the main interface is accessible from any device with a web browser. Additionally, add-on accessories such as wall panels and fader wings can be connected to the network and act as additional inputs.
